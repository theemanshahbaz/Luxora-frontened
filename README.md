Luxora - Premium Luxury Fashion Store 👑
Luxora is a high-end e-commerce frontend project designed for a premium shopping experience. Built with a focus on Glassmorphism UI and smooth performance, it showcases a collection of 40+ luxury items.

🚀 Key Features
40+ Luxury Items: Includes categories like Dresses, Watches, Jewelry, and Perfumes.

Fully Functional Cart: Add items, manage quantities, and see real-time price updates.

Admin Dashboard: A password-protected panel to track revenue and orders.

Modern Aesthetics: Clean, ultra-realistic visual design using Tailwind CSS.

Search Functionality: Real-time search across all luxury products.

💻 Tech Stack
Frontend: HTML5 & Tailwind CSS.

Interactivity: Alpine.js (Lightweight logic).

Icons: FontAwesome.

🛠️ Project Goals
Transitioning from hardcoded data to a MERN stack (MongoDB, Express, React, Node.js) backend.

Rebuilding components using Angular and TypeScript to meet internship requirements.
